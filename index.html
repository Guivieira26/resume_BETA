Quem Eu Sou ?


Olá! Sou Guilherme Henrique Vieira Nascimento, estudante de Ciência da Computação na PUC-MG (Pontifícia Universidade Católica de Minas Gerais) , onde comecei minha jornada acadêmica em 2022. Desde pequeno, minha curiosidade por entender o funcionamento de softwares me motivou a explorar o universo da tecnologia, o que despertou minha paixão pela programação.

Atualmente, busco ampliar meu conhecimento em diversas áreas para encontrar a que mais combina com minhas habilidades e interesses. Tenho afinidade com linguagens back-end como C, Java, SQL e JavaScript, que domino com mais confiança. Também estou me aprofundando em Python, C# e C++ para expandir minhas capacidades.

No desenvolvimento front-end, tenho experiência com HTML/CSS para aplicações web e Flutter para projetos mobile. Também utilizo frameworks como Vue.js, Bootstrap e React, que ajudam a padronizar processos e me tornam apto a desenvolver soluções empresariais robustas e modernas

Estou sempre em busca de novos desafios e oportunidades para aplicar meus conhecimentos e crescer como desenvolvedor.

Minhas Experiências

Iniciei meu estágio em julho de 2023, integrando uma equipe de analistas de TI, onde adquiri uma sólida experiência em infraestrutura de redes e sistemas administrativos. Trabalhei diretamente com a configuração e manutenção de softwares utilizados pelos usuários, utilizando scripts em Shell e ferramentas como Git para automatizar processos e manter os sistemas sempre atualizados.

Meus Projetos


2025 / Jobber Aplicativo Mobile


Desenvolvi um aplicativo em Flutter chamado Jobber , que tem como objetivo conectar pessoas que buscam trabalho com aquelas que desejam contratar. O app permite que os usuários publiquem vagas, exibam seus portfólios e se destaquem no mercado.

Durante o desenvolvimento, aprofundei meus conhecimentos em Flutter, utilizando o Android Studio para compilar e testar no emulador Android. Um dos maiores desafios foi a implementação do backend, onde criamos um sistema de cadastro de usuários com operações CRUD. Para isso, utilizamos JavaScript e JSON para as rotas da API, conectando ao banco de dados MongoDB.

Esse projeto foi uma experiência incrível, consolidando meu aprendizado em desenvolvimento mobile e backend.


2024 / Hashing


Este projeto implementa um sistema de registro de usuários utilizando JPanel para a interface gráfica e armazenando os dados em um arquivo userHash.csv. A senha dos usuários é protegida por hash, garantindo maior segurança.

O desenvolvimento deste projeto surgiu como um experimento para aplicar conceitos aprendidos em Algoritmos e Estruturas de Dados. No entanto, planejo expandi-lo com diversas funcionalidades adicionais.


2024 / Conversor de Bases


Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, implementando um conversor de bases numéricas em Java. O conversor permite a conversão entre binário, decimal e hexadecimal , possibilitando a transformação de qualquer uma dessas bases para as outras.

O desenvolvimento surgiu como parte de um exercício proposto pelo professor , sendo uma excelente oportunidade para consolidar conhecimentos sobre representação numérica, manipulação de strings e operações matemáticas.


2022 / Auto Uploader GitHub


Este projeto foi um dos meus primeiros desenvolvidos em C, criado com o objetivo de automatizar o processo de upload de projetos para o GitHub. No início da minha jornada na programação, percebi a necessidade de executar manualmente os comandos do Git sempre que precisava enviar alterações, então desenvolvi este script para facilitar esse processo.

Com o tempo, passei a utilizar os comandos manualmente devido ao uso de branches, merges e outras funcionalidades do Git , mas este projeto foi uma experiência valiosa , que me ajudou a consolidar conhecimentos sobre sistemas de versionamento e execução de comandos via terminal.

Este projeto representa um marco no meu aprendizado, mostrando minha evolução na programação e no uso do Git.
